# Researcher en Copilot 

## Introducción

Este documento resume el contexto sobre el uso del agente **Researcher** dentro de **Microsoft Copilot** y **Microsoft 365 Copilot**, incluyendo:

* Diferencias entre Copilot personal y Copilot empresarial
* Dónde aparece el agente Researcher
* Requisitos de licencia
* Cómo utilizarlo
* Qué esperar según el tipo de cuenta

---

# 1. Ecosistema de Copilot

Existen dos experiencias principales:

## Copilot para consumidores

Disponible en:

* https://copilot.microsoft.com
* Aplicación de escritorio de Copilot
* Aplicaciones móviles

Características:

* Chat con IA
* generación de imágenes
* respuestas generales
* herramientas experimentales (Labs)

Este entorno **no incluye normalmente los agentes avanzados**.

---

## Microsoft 365 Copilot

Disponible dentro de:

https://microsoft365.com/chat

Este entorno está integrado con herramientas de productividad como:

* Word
* Excel
* PowerPoint
* Outlook
* Teams
* OneDrive

Aquí es donde aparecen funcionalidades avanzadas como **Agents**.

---

# 2. Agentes en Microsoft 365 Copilot

Dentro de Microsoft 365 Copilot existe una sección llamada:

Agents

Ejemplos de agentes:

* Researcher
* Analyst

Estos agentes realizan tareas complejas y especializadas.

---

# 3. Qué es Researcher

Researcher es un agente de IA diseñado para realizar **investigaciones estructuradas automáticamente**.

Capacidades:

* búsqueda de información
* análisis de múltiples fuentes
* generación de reportes
* organización del conocimiento

El resultado suele incluir:

* resumen ejecutivo
* análisis
* tendencias
* fuentes citadas
* recomendaciones

---

# 4. Dónde aparece Researcher

Researcher normalmente aparece en:

https://microsoft365.com/chat

Menú lateral:

Agents

* Researcher
* Analyst

No suele aparecer en:

https://copilot.microsoft.com

aunque la interfaz pueda ser similar.

---

# 5. Requisitos de licencia

Para utilizar Researcher normalmente se requiere:

Licencia de **Microsoft 365 Copilot empresarial**

Ejemplos de planes compatibles:

* Microsoft 365 E3 + Copilot
* Microsoft 365 E5 + Copilot
* Microsoft 365 Business + Copilot

Los planes personales generalmente **no incluyen Researcher**.

Ejemplos:

* Microsoft 365 Personal
* Microsoft 365 Family
* Copilot Pro

---

# 6. Aplicación vs Web

Copilot puede utilizarse desde:

* Web
* Aplicación de escritorio
* Aplicaciones móviles

Sin embargo:

**La disponibilidad de Researcher depende de la licencia, no de la aplicación.**

Es decir:

Si la cuenta no tiene acceso, **Researcher no aparecerá ni en la web ni en la app**.

---

# 7. Cómo usar Researcher

Una vez disponible, el flujo típico es:

1. Abrir Microsoft 365 Copilot
2. Ir a la sección **Agents**
3. Seleccionar **Researcher**
4. Escribir una tarea de investigación

Ejemplo de prompt:

Investiga las tendencias actuales de inteligencia artificial en educación superior.
Incluye herramientas utilizadas, impacto en productividad, ejemplos de universidades y fuentes.

El agente realizará:

* búsqueda
* análisis
* síntesis de información
* generación de informe

---

# 8. Señales para saber si tu cuenta lo tiene

Indicators comunes:

* aparece la sección **Agents**
* se puede seleccionar **Researcher**
* el menú lateral muestra agentes especializados

Si el menú solo muestra:

* Nuevo chat
* Biblioteca
* Tareas
* Descubrir
* Imaginar
* Laboratorios

entonces **Researcher probablemente no está habilitado**.

---

# Conclusión

Researcher es una funcionalidad avanzada de investigación dentro de **Microsoft 365 Copilot** diseñada para automatizar análisis complejos y generación de reportes.

Puntos clave:

* Está orientado a entornos empresariales.
* Se encuentra en **microsoft365.com/chat**.
* Requiere licencia de **Microsoft 365 Copilot**.
* No depende de usar web o aplicación.

---

# Uso potencial en formación

Researcher puede utilizarse para:

* investigación técnica
* preparación de clases
* análisis de mercado
* elaboración de informes
* generación de documentación

Esto lo convierte en una herramienta útil en contextos de **formación tecnológica y productividad con IA**.

# Resumen del módulo AB-730 

## Introducción

Este documento resume el contenido analizado del módulo relacionado con el curso **AB-730T00-A: Transformación de flujos de trabajo empresariales con IA generativa**.

El objetivo del módulo es mostrar cómo **Microsoft 365 Copilot** ayuda a profesionales de negocio a mejorar su productividad mediante el uso de inteligencia artificial integrada en herramientas de trabajo cotidiano.

El enfoque del curso **no es técnico**, sino orientado a **productividad empresarial y uso estratégico de IA generativa**.

---

# Objetivo del módulo

Aprender a utilizar Copilot para:

* Crear contenido profesional
* Investigar información relevante
* Generar presentaciones
* Comunicar resultados

Todo dentro del ecosistema de **Microsoft 365**.

---

# Flujo de trabajo enseñado en el módulo

El módulo sigue un **escenario empresarial típico** donde un profesional debe preparar un informe, una presentación y comunicar resultados a liderazgo.

El flujo completo es:

```
Investigación → Redacción → Visualización → Presentación → Comunicación
```

---

# 1. Investigación con el agente investigador

El **Researcher Agent (Agente Investigador)** es una funcionalidad avanzada dentro de Copilot que permite realizar investigaciones más profundas.

## Funciones principales

* Recopilar información
* Analizar datos
* Generar informes estructurados
* Incluir citas y fuentes

## Fuentes de información

Copilot puede utilizar dos tipos de datos:

### Datos internos (Work data)

Información dentro de la organización:

* Documentos
* Correos electrónicos
* Reuniones
* Chats

### Datos externos (Web data)

Información pública obtenida desde la web.

## Ejemplo de prompt

```
Investigue las últimas tendencias de IA en el sector retail y genere un informe de análisis de mercado.
```

El agente investigador puede:

* hacer preguntas aclaratorias
* generar resúmenes estructurados
* preparar contenido listo para usar

---

# 2. Redacción de contenido con Copilot

Copilot puede utilizarse dentro de **** para crear documentos de forma más eficiente.

## Capacidades

* Generar borradores iniciales
* Mejorar el tono del texto
* Reescribir contenido
* Agregar información relevante
* Organizar ideas

## Ejemplo de prompts

```
Escriba una introducción profesional para nuestro informe de rendimiento del tercer trimestre.
```

```
Haga que el tono sea más optimista y con visión de futuro.
```

## Concepto clave

**Iteración de prompts**

Los usuarios pueden mejorar el resultado refinando continuamente las instrucciones dadas a Copilot.

---

# 3. Visualización de datos

Copilot también permite transformar información textual en estructuras visuales.

## Ejemplo

Convertir texto en una tabla.

Proceso:

1. Seleccionar un párrafo
2. Pedir a Copilot que lo convierta en tabla
3. Refinar el formato con prompts adicionales

Ejemplo:

```
Convierte este párrafo en una tabla con columnas para:
Equipo | KPI | Resultado
```

---

# 4. Creación de presentaciones

Copilot puede utilizarse dentro de **PowerPoint** para generar diapositivas automáticamente.

## Flujo típico

1. Crear informe en Word
2. Usar Copilot para resumir el documento
3. Generar diapositivas para presentación

Ejemplo de prompt:

```
Resume este contenido en cinco puntos de viñetas para una presentación de liderazgo.
```

También se pueden refinar los resultados:

```
Haz que los puntos sean más concisos y orientados a la acción.
```

---

# 5. Comunicación de resultados

Finalmente, Copilot puede utilizarse en **Outlook** para comunicar resultados al equipo o liderazgo.

## Capacidades

* Resumir hilos de correos electrónicos
* Redactar correos profesionales
* Generar actualizaciones ejecutivas

Ejemplo:

```
Redacta un correo al equipo directivo con el resumen del informe.
```

# Conceptos clave del módulo

Los principales conceptos que se evalúan en este contenido son:

## 1. Prompting efectivo

Formular instrucciones claras para obtener mejores resultados de la IA.

## 2. Iteración de prompts

Refinar progresivamente las instrucciones para mejorar la salida.

## 3. Uso de fuentes de información

Combinar:

* Datos internos de la organización
* Datos externos de la web

## 4. Integración entre aplicaciones

Copilot funciona dentro de varias herramientas del ecosistema Microsoft 365:

* Word
* PowerPoint
* Outlook

Esto permite completar flujos de trabajo completos sin cambiar de entorno.

# Diferencia importante

Copilot tiene diferentes funcionalidades con objetivos distintos.

| Función          | Propósito                                       |
| ---------------- | ----------------------------------------------- |
| Copilot Chat     | Redacción y edición de contenido                |
| Researcher Agent | Investigación profunda y generación de informes |

# 📊 Microsoft 365 Copilot – Análisis de Datos con Excel y el Agente Analyst

## 🧭 Contexto del módulo

En el entorno empresarial actual, **la toma de decisiones basada en datos** es fundamental. Sin embargo, muchas personas no tienen tiempo ni conocimientos técnicos avanzados para analizar grandes conjuntos de datos.

Para resolver esto, **Microsoft 365 Copilot** incorpora herramientas de **inteligencia artificial** que ayudan a transformar datos en información útil de manera rápida.

En este módulo se presentan dos herramientas principales:

- **Copilot en Excel**
- **Agente Analyst**

El objetivo es aprender a pasar de **datos sin procesar → insights → reportes listos para compartir**.

---

# 🧩 Herramientas principales

## 1️⃣ Copilot en Excel

Copilot integrado en **Microsoft Excel** permite analizar datos usando **lenguaje natural**.

El usuario puede hacer preguntas o dar instrucciones como si estuviera hablando con un analista.

Ejemplos de tareas que Copilot puede realizar:

- resumir datos
- detectar tendencias
- encontrar valores atípicos
- generar gráficos
- crear fórmulas
- explicar fórmulas existentes
- importar datos desde la web

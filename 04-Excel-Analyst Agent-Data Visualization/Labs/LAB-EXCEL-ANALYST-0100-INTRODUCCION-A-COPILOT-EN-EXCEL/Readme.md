# LAB-EXCEL-ANALYST-0100-INTRODUCCION-A-COPILOT-EN-EXCEL

## Objetivo
Familiarizarse con **Microsoft 365 Copilot en Excel**, su interfaz y funcionalidades básicas, para poder interactuar con datos mediante prompts. Al finalizar, el alumno podrá:
- Localizar el panel de Copilot Chat en Excel.  
- Escribir prompts simples y obtener respuestas automáticas.  
- Comprender la diferencia entre Copilot Chat y el agente Analyst.  

---

## Requisitos previos
- Cuenta de Microsoft 365 con Copilot habilitado.  
- Acceso a Excel en la web o en escritorio.  

---

## Paso 1: Ingresar a Microsoft 365
1. Abrir el navegador web.  
2. Ir a [www.microsoft365.com](https://www.microsoft365.com).  
3. Iniciar sesión con tu cuenta de Microsoft 365.  
4. Seleccionar **Excel** desde el iniciador de aplicaciones.  

---

## Paso 2: Crear un nuevo archivo de Excel
1. Hacer clic en **Nuevo libro en blanco**.  
2. Guardar el archivo en **OneDrive** (Copilot necesita que el archivo esté en la nube).  
   - Ejemplo de nombre: `Lab_Intro_Copilot.xlsx`.  

---

## Paso 3: Abrir Copilot Chat
1. En la barra superior de Excel, buscar y hacer clic en el **botón Copilot**.  
2. Se abrirá el **panel Copilot Chat** en la parte derecha.  
3. Familiarizarse con la interfaz:  
   - Área para escribir prompts.  
   - Panel de resultados con resúmenes, sugerencias y gráficos generados por Copilot.  

---

## Paso 4: Explorar prompts básicos
### 4.1 Prueba de texto simple
- Escribir en Copilot Chat:  
```

Explícame qué es este archivo de Excel

```
- Revisar la respuesta generada y observar cómo Copilot interpreta los datos.

### 4.2 Preguntas de ejemplo
- Preguntar cosas simples como:  
```

Dame un resumen de los datos

```
```

¿Cuáles son los valores máximos y mínimos de cada columna?

```
- Observar cómo Copilot genera información sin necesidad de fórmulas.  

### 4.3 Prompts para visualización
- Escribir:  
```

Crea un gráfico de barras de ejemplo con estos datos

```
- Nota: si no hay datos cargados aún, Copilot generará un gráfico de ejemplo o indicará que agregues datos.  

---

## Paso 5: Diferencias entre Copilot Chat y el agente Analyst
- **Copilot Chat:** interactúa con los datos del archivo actual, genera resúmenes, gráficos y sugerencias.  
- **Agente Analyst:** disponible solo en cuentas empresariales/educativas con licencia Copilot; permite análisis más avanzado, razonamiento de cadena, comparaciones entre archivos y generación de reportes automáticos.  
- Este lab se concentra **solo en Copilot Chat**.  

---

## Paso 6: Recomendaciones
- Escribir prompts claros y específicos para obtener mejores resultados.  
- Explorar varias formas de preguntar sobre el mismo archivo para familiarizarse con la herramienta.  
- Guardar cambios continuamente en OneDrive.  
- No es necesario tener datos complejos; este lab sirve para **practicar interacción con Copilot**.  

---

## Conclusión
Al finalizar este laboratorio, el alumno habrá aprendido a:  
- Abrir Copilot Chat en Excel.  
- Hacer prompts simples para generar resúmenes y visualizaciones.  
- Entender la interfaz y funcionalidades básicas de Copilot en Excel.  

Este lab sirve como **base para futuros laboratorios**, donde se trabajará con **exploración de datos, gráficos y análisis avanzado con el agente Analyst**.

# LAB-EXCEL-ANALYST-0300-IMPORTAR-DATOS-DESDE-LA-WEB-CON-COPILOT

## Objetivo
Aprender a importar datos desde la web usando **Copilot Chat en Excel**, colocando la información en hojas nuevas y preparándola para análisis posterior.

---

## Requisitos previos
- Cuenta de Microsoft 365 con Copilot habilitado.  
- Acceso a Excel en la web o escritorio.  
- Guardar archivos en OneDrive.

---

## Paso 1: Crear un nuevo archivo de Excel
1. Abrir Excel desde [www.microsoft365.com](https://www.microsoft365.com).  
2. Seleccionar **Nuevo libro en blanco**.  
3. Guardar el archivo en **OneDrive** con un nombre como: `Lab_Importar_Datos.xlsx`.  

---

## Paso 2: Abrir Copilot Chat
1. Hacer clic en el botón **Copilot** en la barra superior de Excel.  
2. Se abrirá el panel **Copilot Chat** a la derecha.  

---

## Paso 3: Crear una hoja con países de Latinoamérica
1. Crear una **nueva hoja**.  
2. En la columna A poner el encabezado: `País`.  
3. Escribir en Copilot Chat:  
```

Completame la columna A paises con la lista de paises de LATAM ordenados alfabeticamente

```
4. Copilot completará automáticamente la columna con los 21 países de Latinoamérica ordenados alfabéticamente, aplicando formato en negrita al encabezado y ajustando el ancho de la columna.  

---

## Paso 4: Completar la columna de capitales
1. En la columna B poner el encabezado: `Capital`.  
2. Escribir en Copilot Chat:  
```

Completame la columna B de la hoja actual con la lista de capitales de los paises de la columna A

```
3. Copilot completará la columna B con las capitales correspondientes, aplicando formato en negrita al encabezado y ajustando el ancho de la columna.  

---

## Paso 5: Importar datos desde la web
1. Crear una **nueva hoja** para los datos externos.  
2. Seleccionar el icono de **globo / contenido conectado a la web** en el panel Copilot Chat.  
3. Escribir un prompt como ejemplo:  
```

Busca en la web una tabla de países y sus tipos de cambio

```
4. Copilot buscará los datos y generará la tabla en una **nueva hoja** de Excel.  
5. Revisar los datos importados y validar que se vean correctos.  

---

## Paso 6: Preparar los datos para análisis
1. Convertir cada rango de datos en **tabla de Excel**: `Insertar → Tabla`.  
2. Verificar encabezados y formatos: fechas, números y texto.  
3. Guardar cambios en OneDrive.  

---

## Paso 7: Exploración rápida (opcional)
1. Probar prompts simples con Copilot para validar los datos, por ejemplo:  
```

Resume la cantidad de países y sus tipos de cambio

```
2. Observar cómo Copilot interpreta y organiza la información.  

---

## Conclusión
Al finalizar este laboratorio, el alumno será capaz de:  
- Crear hojas nuevas y usar encabezados claros.  
- Importar datos desde la web con Copilot Chat.  
- Completar columnas automáticamente con listas predefinidas o información externa.  
- Preparar los datos para análisis y visualización en Excel.

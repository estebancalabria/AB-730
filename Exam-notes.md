* Apps
  * Excel
    * Copilot permite cosas como "Generá un resumen de los puntos clave de tus datos." o "Crear una tabla pivote basada en tus datos"
  * Word
    * Copilot permite insertar una tabla de contenido basado en tu documento o "Generá un resumen de los puntos clave de tus datos."
  * Outlook
    * Copilot permite crear un borrador de respuesta para un mail
    * Copilot permite hacer un resumen de los mails sin leer en tu inbox
  * Power Point
    * Copilot puede Create a slide layout based on the content of your presentation.
    * Copilot puede Produce a summary slide based on the key points in your presentation.
  * Teams
    * Copilot puede Create a meeting summary, including key decisions and action items.
    * Copilot puede Draft a response to a channel message based on the context of the conversation.
    * Copilot no puede mandar mails por si solo
  * Copilot Chat
    * Podes borrar una conversacion individual y no lo pueden prohibir

* Security 
    * Si me comparten un prompt y no lo puedo ver es qeu fue hecho fuera de mi organizacion
    * Si me dice algo como "You asked for a summary of File.docx. However, the file appears to be either empty, corrupted, or in a format that I cannot process." es que no tengo acceso al archivo
    * Para chequear si una imagen fue hecha con copilot tengo que ver los "content credentials" (https://contentcredentials.org/verify)
    * Copilot provides responses based only on data that the user can access and the general knowledge that Copilot was trained on.
    * Copilot NO SE ENTRENA con datos de tu organizacion (si puede acceder a ellos)
    * Copilot respeta todas las politicas de seguridad definidas para tu organizacion en 365

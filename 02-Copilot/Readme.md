# Microsoft 365 Copilot & Copilot Chat

Este repositorio explica las principales funcionalidades de **Microsoft 365 Copilot**, incluyendo **Copilot Chat**, creación y gestión de **agentes de IA**, administración de mensajes y conversaciones, y buenas prácticas para avisos y consultas.

---

## 1. ¿Qué es Microsoft 365 Copilot?

Microsoft 365 Copilot es un asistente de inteligencia artificial integrado en aplicaciones como Word, Excel, PowerPoint, Outlook y Teams. Su objetivo es:

- Automatizar tareas repetitivas.
- Resumir, redactar y editar contenido.
- Integrar información de distintas aplicaciones y fuentes.

Copilot combina modelos de lenguaje grandes (LLM, como GPT-4 y GPT-5) con datos empresariales seguros, permitiendo respuestas precisas y contextualizadas.

---

## 2. Copilot Chat

**Copilot Chat** es la experiencia conversacional de Copilot, diseñada para interactuar con varias aplicaciones de manera fluida.

### Características principales

- Chat conversacional con contexto continuo.
- Disponible en aplicaciones de Microsoft 365, Teams o navegador.
- Acceso seguro a datos de trabajo y web.
- Modos de funcionamiento:
  - **Trabajo:** utiliza información interna de la organización (correos, archivos, reuniones) para tareas directas del trabajo.
  - **Web:** obtiene información pública de la web para investigación, brainstorming y actualización de tendencias.

### Funcionalidades clave

1. **Memoria de Copilot:** recuerda preferencias y contexto de trabajo para respuestas consistentes.
2. **Búsqueda inteligente:** localiza información relevante en documentos y correos mediante Microsoft Graph.
3. **Control de orígenes:** permite alternar entre datos de trabajo y web.
4. **Entrada de voz:** dictado, lectura en voz alta y chat de voz para mayor accesibilidad.

---

## 3. Escritura de avisos efectivos

Para maximizar el valor de Copilot:

- Sea claro, específico y conversacional.
- Proporcione contexto, objetivos y expectativas.
- Incluya referencias directas a archivos (`/Archivo`), temas (`#Tema`) o personas (`@Nombre`).
- Itere y refine las consultas según los resultados.
- Evite lenguaje confuso o instrucciones contradictorias.

### Ejemplos de uso

- Redacción de correos electrónicos.
- Preparación de presentaciones.
- Planificación de reuniones y gestión del tiempo.
- Análisis de datos y visualización de tendencias.

## 4. Administración de mensajes y conversaciones

Copilot Chat permite organizar los mensajes y conversaciones para un flujo de trabajo eficiente:

### Mensajes

- **Guardar:** almacenar mensajes frecuentes en la Galería de consultas.
- **Programar:** automatizar tareas en horarios específicos (máx. 10 mensajes programados).
- **Compartir:** distribuir mensajes guardados con otros miembros del equipo.

### Conversaciones

- Buscar chats anteriores usando palabras clave.
- Cambiar el nombre de los chats para mejor organización.
- Eliminar conversaciones irrelevantes.
- Guardar conversaciones en cuadernos de Copilot para análisis y seguimiento.

## 5. Agentes de Copilot

Los **agentes** son asistentes de IA especializados dentro de Copilot Chat, diseñados para tareas específicas.

### Tipos de agentes

1. **Precompilados:** listos para usar, como:
   - **Investigador:** recopila y resume información.
   - **Analista:** interpreta datos y genera visualizaciones.
2. **Personalizados:** creados para flujos de trabajo específicos de la organización.

### Crear un agente

1. Abrir Copilot Chat → **Crear agente**.
2. Describirlo usando plantilla o lenguaje natural.
3. Configurar:
   - Nombre, descripción e ícono.
   - Instrucciones: rol, tareas y estilo de respuesta.
   - Conocimiento: hasta 20 orígenes (SharePoint, OneDrive, Teams, web).
   - Capacidades: intérprete de código, creador de imágenes.
4. Agregar avisos sugeridos para orientar a los usuarios.

### Compartir un agente

- Mantiene privacidad hasta que decidas compartirlo.
- Opciones: usuarios/grupos específicos o toda la organización.
- Roles: **Visor** (uso) y **Editor** (modificación).
- Nota: cambios en mensajes sugeridos requieren reinstalación del agente para reflejarse en otros usuarios.
